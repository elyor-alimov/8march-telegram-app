# 🌸 8 Марта — Telegram Mini App

Telegram Mini App для праздника **8 Марта (Международный женский день)**. Это одностраничное веб-приложение с уникальным дизайном «Барби × Майнкрафт» — яркие розовые и фиолетовые тона с пиксельным шрифтом и плавными анимациями.

## 📱 Что умеет приложение

- **🌐 Двуязычное** — Русский 🇷🇺 и Узбекский 🇺🇿
- **💌 101 причина** — 101 карточка с комплиментами, которые переворачиваются по нажатию
- **🎁 Подарочная коробка** — 9 сюрпризов (8 цветов + 1 видео)
- **🐱 Kitty Runner** — мини-игра на canvas: котик бежит и собирает монетки
- **🏪 Магазин призов** — обменяй монетки на реальные подарки
- **🎵 Фоновая музыка** — переключается кнопкой
- **📊 Google Sheets трекинг** — результаты игры и заявки на призы

## 🖼️ Раздел 1: Подготовка картинок

### Спрайты котика (4 файла)
- `kitty_run1.png`, `kitty_run2.png`, `kitty_run3.png` — 3 кадра бега
- `kitty_jump.png` — кадр прыжка
- **Размер холста**: 100×100 пикселей
- **Котик**: по центру ~80×80 пикселей, 10 пикселей прозрачного отступа со всех сторон
- ⚠️ Отступы нужны, чтобы котик стоял на земле, а не «летал» над ней

### Спрайты монстров (6 файлов)
- `monster1_f1.png`, `monster1_f2.png`, `monster1_f3.png` — монстр типа 1 (3 кадра)
- `monster2_f1.png`, `monster2_f2.png`, `monster2_f3.png` — монстр типа 2 (3 кадра)
- **Размер холста**: 60×80 пикселей
- **Монстр**: по центру ~44×64 пикселей с 8px отступами

### Цветы (8 файлов)
- `flower1.png` ... `flower8.png`
- Любой размер (в приложении показываются как 48×48)
- Цветы: тюльпан, роза, ромашка, ландыш, пион, подсолнух, фиалка, жасмин

> 💡 **Если у вас нет картинок** — приложение всё равно будет работать! Оно покажет эмодзи вместо картинок.

## 🎬 Раздел 2: Видео и музыка

### Видео-сюрприз
1. Положите любое видео в папку `assets/video/`
2. Назовите его точно `cat_salute.mp4`
3. Это видео покажется при открытии 9-й подарочной коробки

### Фоновая музыка
1. Положите любой MP3 файл в папку `assets/music/`
2. Назовите его точно `bg.mp3`
3. Музыка включается кнопкой в главном меню

## 🚀 Раздел 3: Загрузка на GitHub Pages (пошагово)

### Шаг 1: Создать аккаунт на GitHub
1. Откройте сайт **github.com**
2. Если нет аккаунта — нажмите **Sign up** и зарегистрируйтесь
3. Подтвердите email

### Шаг 2: Создать новый репозиторий
1. Нажмите зелёную кнопку **"New"** (или **"+"** справа вверху → **"New repository"**)
2. Введите название, например: `8march-app`
3. Выберите **Public** (Публичный)
4. Нажмите **"Create repository"**

### Шаг 3: Загрузить файлы
1. На странице пустого репозитория нажмите **"uploading an existing file"**
2. Перетащите **ВСЕ** файлы проекта (включая папку `assets`)
3. Ниже нажмите **"Commit changes"**

### Шаг 4: Включить GitHub Pages
1. Перейдите в **Settings** (Настройки) репозитория (вкладка вверху)
2. В левом меню найдите **Pages**
3. В разделе **Source** выберите:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажмите **Save**
5. Подождите 1-2 минуты
6. Появится ссылка: `https://ваш-логин.github.io/8march-app/`

## 🤖 Раздел 4: Создание Telegram бота и Mini App

### Шаг 1: Создать бота
1. Откройте Telegram
2. Найдите бота **@BotFather**
3. Отправьте команду `/newbot`
4. Введите **отображаемое имя** бота (например: «8 Марта 🌸»)
5. Введите **юзернейм** бота — он должен заканчиваться на `bot` (например: `march8gift_bot`)
6. BotFather пришлёт вам **токен** — сохраните его

### Шаг 2: Создать Mini App
1. Снова @BotFather, отправьте `/newapp`
2. Выберите своего бота из списка
3. Введите название приложения
4. Введите описание
5. Отправьте картинку-превью (640×360)
6. Когда попросят **Web App URL** — вставьте вашу ссылку с GitHub Pages
7. Готово! Бот подключён к приложению

## 📊 Раздел 5: Настройка Google Sheets трекинга

### Шаг 1: Создать Google Таблицу
1. Откройте **Google Таблицы** (sheets.google.com)
2. Создайте **пустую таблицу**

### Шаг 2: Добавить Apps Script
1. В таблице: **Расширения** → **Apps Script**
2. Удалите весь код, который там есть
3. Скопируйте **весь** код из файла `google_apps_script.js` и вставьте
4. Нажмите **Сохранить** (или Ctrl+S)

### Шаг 3: Развернуть скрипт
1. Нажмите **Развернуть** → **Новое развёртывание**
2. Тип: **Веб-приложение**
3. Описание: любое (например «8 march tracker»)
4. Выполнять от: **Я**
5. Доступ: **Все**
6. Нажмите **Развернуть**
7. Скопируйте **URL развёртывания**

### Шаг 4: Подключить трекер
1. Откройте файл `tracker.js`
2. Найдите строку: `var TRACKER_SCRIPT_URL = '';`
3. Вставьте URL из шага 3 между кавычками
4. Загрузите обновлённый `tracker.js` на GitHub

## 🔧 Раздел 8: FAQ — Частые вопросы

**Видео не воспроизводится?**
→ Проверьте, что файл называется точно `cat_salute.mp4` и лежит в `assets/video/`

**Музыка не включается?**
→ Это нормально! Браузеры блокируют автовоспроизведение. Пользователь должен нажать кнопку «Вкл. музыку»

**Google Таблица пустая?**
→ Проверьте, что URL в `tracker.js` правильный и скрипт развёрнут с доступом «Все»

**Хочу изменить причины?**
→ Откройте `reasons.js`, замените текст в массивах `REASONS_RU` и/или `REASONS_UZ`

**Хочу изменить призы?**
→ Откройте `i18n.js`, найдите массив `prizes` и измените названия, места и стоимость

**Приложение работает в браузере, но не в Telegram?**
→ Проверьте, что правильный URL указан в BotFather при создании Mini App

**Как сбросить прогресс?**
→ В BotFather перейдите к боту и очистите localStorage через консоль браузера:
```
localStorage.clear()
```

## 💻 Технические детали

- **Чистый HTML/CSS/JS** — без фреймворков и сборщиков
- **Шрифт**: Press Start 2P (пиксельный)
- **Цвета**: фиолетовый (#1A0030 → #4A1570), розовый (#E91E8C), золотой (#FFD700)
- **localStorage**: сохраняет язык, имя, монетки, открытые карточки
- **Canvas игра**: физика прыжков, спрайтовая анимация, частицы, нарастающая сложность
